1. Мини-компрессор строки (RLE)
Реализовать сжатие строки по принципу Run-Length Encoding:
AAABBCCDAA → A3B2C2D1A2
и обратную операцию — декодирование.
Нельзя использовать Regex.Replace, только логика + StringBuilder.

2. Удаление всех вложенных скобок (Regex + StringBuilder)
Дана строка с любыми вложенными скобками:
"a(bc(de)f)g(h(i))"
Нужно удалить только содержимое скобок, сами скобки тоже убрать:
Вывод: "ag"
Сложность — корректно удалять вложенные уровни.

3. Проверка корректности арифметического выражения
Дано выражение вида:
"(2+3)*(4-(5/2))+7"
Необходимо с помощью Regex + логики проверить:
корректность скобок (вложенность)
корректность операторов (+ - * /)
отсутствие двух операторов подряд
отсутствие оператора в конце
Regex будет частичным решением — это сложная комбинация логики и регулярных выражений.

1) Скопировать код класса Task в Program.cs
2) Запустить программу

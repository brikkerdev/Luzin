# Лабораторная работа 3: Реализация коллекций и интерфейсов

## Цель работы
Познакомиться с интерфейсами коллекций в C# (.NET), реализовать простые коллекции и понять их контракт и функциональность.

## Реализованные коллекции

### 1. SimpleList
Реализует интерфейсы `IEnumerable`, `ICollection`, `IList`. Внутренне использует динамический массив (без готовых коллекций List и т.п.).

**Основные возможности:**
- Добавление элементов (`Add`, `Insert`)
- Удаление элементов (`Remove`, `RemoveAt`, `Clear`)
- Поиск элементов (`Contains`, `IndexOf`)
- Индексированный доступ через `[]`
- Поддержка перебора через `foreach`
- Автоматическое увеличение емкости при необходимости

**Реализованные интерфейсы:**
- `IEnumerable` - поддержка перебора элементов
- `ICollection` - базовые операции: добавление, удаление, подсчёт элементов
- `IList` - индексированный доступ, вставка, удаление по индексу

### 2. SimpleDictionary<TKey, TValue>
Реализует `IDictionary<TKey, TValue>` и `IReadOnlyDictionary<TKey, TValue>`. Использует хеш-таблицу с цепочками для разрешения коллизий.

**Основные возможности:**
- Добавление пар ключ-значение
- Поиск по ключу (`ContainsKey`, `TryGetValue`)
- Удаление по ключу или по паре
- Перебор всех элементов
- Автоматическое рехеширование при заполнении
- Поддержка кастомного компаратора

**Реализованные интерфейсы:**
- `IDictionary<TKey, TValue>` - словарь с парами ключ-значение
- `IReadOnlyDictionary<TKey, TValue>` - доступ только для чтения

### 3. SimpleDoubleLinkedList<T>
Реализует интерфейс `IList<T>` с использованием двунаправленного связного списка.

**Основные возможности:**
- Добавление в начало/конец списка (`AddFirst`, `AddLast`)
- Вставка по индексу
- Удаление из начала/конца списка (`RemoveFirst`, `RemoveLast`)
- Быстрый доступ к элементам с начала или конца
- Эффективная вставка и удаление в середине списка

**Реализованные интерфейсы:**
- `IList<T>` - типизированный список с индексированным доступом
- `ICollection<T>` - базовые операции с элементами

# Лабораторная работа №4
## Проблемы синхронизации потоков

### Цель работы
Изучить проблему синхронизации потоков, научиться выявлять и предотвращать взаимоблокировки (deadlock), а также реализовывать классические задачи синхронизации.

### Краткое описание задач

- Задача 1: Обедающие философы (Dining Philosophers)  
  Демонстрация взаимоблокировки при неправильном порядке захвата ресурсов и её устранения с помощью смены порядка и семафоров.

- Задача 2: Спящий парикмахер (Sleeping Barber)  
  Моделирование работы парикмахерской с ограниченной очередью клиентов с использованием семафоров и блокировок.

- Задача 3: Производитель–Потребитель (Producer–Consumer)  
  Взаимодействие производителей и потребителей через общий ограниченный буфер с классической схемой трёх синхронизирующих примитивов.

### Как запустить

В `Program.cs` вызвать метод `Main`

Подробнее о каждой задаче см. в соответствующих README в папках Task1, Task2 и Task3.
